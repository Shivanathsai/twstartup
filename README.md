## Initial Setup

```
$ git clone git@github.com:TWstartups/twstartups.github.io.git
$ cd twstartups
$ ./scripts/setup.sh
```

## Develop

```
$ cd client
$ npm run dev
<!-- go to http://localhost:3000/ -->
```

```
$ cd server
$ npm run dev
<!-- go to http://localhost:8000/ -->
```
